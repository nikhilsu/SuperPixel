# SuperPixel
Replicating the Super Resolution paper - https://arxiv.org/pdf/1707.00737.pdf

#### Dependencies
- ```OpenCV```
- ```Keras```
- ```Numpy```

##### Prior to training
- Ensure the [dataset](https://www.google.com) is downloaded into the project directory
- The directory structure should look as follows
```
SuperPixel
|
|--- dataset
      |
      |---- 10000.npy
```

##### Train Model

```bash
$ python main.py
```
